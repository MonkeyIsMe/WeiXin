中南大学信息科学与工程学院信息安全专业实习实训方案

一、实训目的

为使学员了解网络安全相关岗位技能，让学员能够实际体验安全问题的分析思路和分析过程， 在工作实践中能直接应用操作与演示的内容，促使学员全面提升就业竞争力。

二、实训人员

中南大学信息科学与工程学院信息安全专业大三学生

三、实训时间

2018年8月27日-2018年9月14日（三周）

四、实训地点

湖南合天智汇信息技术有限公司

五、实训条件

1、实训教师、实训场所（办公室、会议室），包括投影仪等，均由乙方负责安排。

2、不安排学员住宿，协助学生中餐预定，费用学生自理。

六、实训形式

1) 根据公司提供的岗位方向让学生事先进行分组，根据问卷意向分为3组成员共计60人，每小组推选1名学生任组长便于小组的日常及行政管理；选出1位学生任总管理人员负责任务下达和数据统计及校企之间的沟通；

2) 每周安排一天上午进行大课程讲解，学生工作内容的相关安排及总结;

3) 针对每个岗位方向进行实训，根据学员意向分配在各个岗位进行学习，除大课程安排学生全员参与外，各组的实训内容导师可根据给各组学生情况进行分配及指导；

4) 每周五上午11点30前给各导师交周报，组长先负责搜集做整体统计后再交于导师，每周五下午每组导师对学生进行评分及意见指导，最后一周周五的总结不需上交。

5) 最后一天下午为整体实训工作总结；

6) 每个学生评分（100分）是由三部分组成：①考勤分共计20分，②每组综合评分60分，③20分整体综合表现分数。

运维组（平台、运维）

第一周

主要内容：

1. 介绍如何在vmware下运行centos7虚拟机

2. 介绍centos7的一些基本操作

3. 介绍KVM的安装与使用，演示已安装好的KVM虚拟机

4. 介绍openstack的简单架构

5. 完成任务课程

课程任务：

1. vmware中虚拟机，都用nat模式网络，系统选择centos7.3

2. Centos7系统创建一个centos用户，配置密码，使用centos登入系统

3. Centos7中安装搭建ftp和http服务器

4. 查看两个服务日至，并将日志复制压缩到http文件目录下。

5. 验证: 打开浏览器, 输入centos虚拟机IP, 能看到压缩日志为成功

6. 再生成一台centos虚拟机，两台虚拟机禁止密码登录，只能秘钥互相登录，并将http压缩文件复制到第二台虚拟机中。

达到效果：

1. 能够使用vmware搭建centos的虚拟环境

2. 能够在centos7上配置网卡，安装软件，管理服务，查看编辑配置文件，查看日志，压缩和解压文件，远程传输文件等一些基本操作

3. 了解KVM搭建虚拟机，并进行拷贝、快照、开机自启等一系列基本操作。

4. 对openstack有初步的了解

第二周

主要内容：

1. 介openstack搭建方法（针对官网）

2. openstack的初步体系了解，并演示已经搭建好的openstack平台

3. 介绍openstack的使用

4. 介绍openstack模板的制作

课程任务：

1. 制作一个centos7的镜像和一个windows7的景象，并且能够成功在云平台中使用

达到效果：

1. 对openstack的大致体系能够了解

2. 对openstack的搭建流程有一个大致的了解

3. 对openstack基础操作有一个大致了解

4. 成功制作openstack模板

第三周

主要内容：

1. 演示openstack创建网络和实例的方法

2. 介绍容器docker及dockerfile搭建部署

课程任务：

1. 在VM上生成centos虚拟机

2. 安装docker组件

3. 更改docker源

4. 下载centos基础image

5. 在本地生成包含hello heetian 的基础txt文件

6. 编写dockerfile, 生成docker容器

7. 容器要求: 包含http服务, 能通过本机浏览器访问容器的http服务. 网页中包含hello heetian 的基础txt文件

达到效果：

1. 了解openstack怎么创建网络和实例

2. 能够使用wmware部署操作系统，搭建docker环境，并能动手编写简单dockerfile，生成docker容器
